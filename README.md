# Calculator

This is a TypeScript simple calculator.

## Installation

To install the project run the following commands:

```
git clone https://github.com/B1lol-dev/ts-calculator
```

```
npm install
```

## Usage

To start the calculator, run the following command:

```
npm run dev
```

## Features

- Addition
- Subtraction
- Multiplication
- Division
