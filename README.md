# Calculator

This is a TypeScript simple calculator.

## Installation

To install the project dependencies, run the following command:

```
npm install
```

## Usage

To start the calculator, run the following command:

```
npm start
```

## Features

- Addition
- Subtraction
- Multiplication
- Division
